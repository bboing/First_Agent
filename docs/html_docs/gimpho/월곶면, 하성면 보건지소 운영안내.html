업무설명

1. 월곶면 보건지소

  - 매주 월,화,수 운영 ( 목,금 미운영)

  - 예방접종 : 직접 유선 연락하여 예약 후 방문

     (백신 수량, 공중보건의 부재 등 문제 될수 있으므로 예약 필수)

  - 내과 진료 : 3월 중 공중보건의 근무 종료 예정으로 유선상으로 일정 상담 후 방문 해야함 

  - 사전연명의료의향서 작성, 치매인지선별검사 : 월,화 9시~4시(점심시간 제외/신분증지참)

2. 하성면 보건지소

  - 매주 목, 금 운영 (월,화,수 미운영)

  - 예방접종 : 직접 유선 연락하여 예약 후 방문

    (백신 수량, 공중보건의 부재 등 문제 될수 있으므로 예약 필수)

  - 내과 진료 :  3월 중 공중보건의 근무 종료 예정으로 유선상으로 일정 상담 후 방문 해야함 

  - 사전연명의료의향서 작성, 치매인지선별검사 : 금요일 9시~4 시(점심시간 제외/신분증지참)

주의사항
특이사항

1인 직원이 월~수 월곶면 보건지소, 목~금 하성면 보건지소 순회 근무

-> 월 ~수는 월곶면으로 유선연결, 목~금은 하성면으로 유선연결 가능
