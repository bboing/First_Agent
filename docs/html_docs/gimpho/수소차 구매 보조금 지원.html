업무명

수소차 구매 보조금 지원

업무개요

수소차(승용, 고상버스 ) 구매 시 보조금을 지원하여 친환경차 보급을 활성화

접수처

기후에너지과

대상자

수소차를 구매하고자 하는 김포시민

( 신청일 기준 3개월 연속하여 김포에 주소를 둔 자)

신고기한

예산 소진 시까지

과태료

해당없음

구비서류

○ 지원신청

  - (공통) 수소차 구매 지원신청서, 차량 구매계약서(출고예정일 기재)

  - (개인) 주민등록초본

  - (개인사업자) 주민등록초본 및 사업자등록증

  - ( 법인, 기업체 등) 법인등기부등본, 사업자등록증

※ 그 외 우선순위, 추가지원 대상의 경우 필요서류 공고문 참고

처리기간

2~3 개월 (보조금 지급 완료까지)

처리비용

없음

신청방법

○ 수소차 제조·수입사에서 시스템 통해 대행접수

  - 신청인(구매자) : 수소차 제조·판매 영업점(전국 가능)을 방문하여 구매계약 체결 및 지원 서류 제출

  - 제조·수입사 : 신청인이 제출한 지원서류를 구매보조금 지원 시스템에 접수

교부방법

대상자선정 후 차량 출고·등록 → 제조·수입사 보조금 지급 신청 → 지급신청 서류검토 후 보조금 지급

처리절차

1. (구매자) 수소차 구매계약 체결 → 2. ( 제조·수입사) 수소차 구매지원 신청서 접수 →

3. (김포시) 신청 검토 후 자격부여 → 4. (제조·수입사) 보조금 지원 가능 확인 요청(10일 내 출고가능 시) →

5. (김포시) 보조금 지원대상자 확정 통보 → 6. (제조·수입사) 차량출고·< span style="line-height: 1.2; font-family: 굴림; font-size: 10pt; margin-top: 0px; margin-bottom: 0px;">등록 후 구매보조금 지급 신청 →

7. (김포시) 보조금 지급

처리방법

무공해차 구매보조금 지원시스템 통해 신 청 접수 및 처리

업무설명

<주요 문의사항>

○ 공고문 확인방법 : 김포시청 홈페이지 고시·공 고 게재

○ 신청방법 : 구매계약 체결 후 전기차 판매·영업점 통해 시스템으로 신청

○ 예산(잔여대수) 확인

  - 무공해차 통합누리집(www.ev.or.kr)접속 < /span>→ 구매 및 지원 → 구매보조금 지급현황 → 김포시(비고참고)

○ 차종별 보조금액 : 수소승용(넥쏘) 32,500천원/대< /span>, 수소고상버스(유니버스 수소전기버스) 350,0 00천원/대

○ 자세한 내용 은 공고문 참고 안내

주의사항
특이사항

○ 매년 2월 말 ~ 3월 초 김포시청 홈페이지 고시·공고에 공고문 게재

○ 차종별 보조금액 : 수소승용(넥쏘) 32,500천원/대, 수소고상버스 (유니버스 수소전기버스) 350,000천원/대

○ 보조금 재지원 제한 : 승용 2년· 버스 2년 내 동일 차종 차량을 구매하는 경우 보조금 미지원

○ 의무사항 (※ 차량 말소·매매에 따른 문의전화는 기후에너지과로 연 결 요청)

- 보조금을 지급받은 전기차의 구매자는 의무운행기간 준수

- 의무운행기간 내 차량말소(폐차2 년, 수출5년) 시 김포시의 사전승인을 받아야 하며, 운행기간별 보조금 환수율에 따라 보조금 반납
