 & nbsp;  3. 공장의 업종(분류번호) : 한국표준산업분류상  업종명& nbsp;및 업종분류번호(5자리) 기재 [통계청 홈페이지(www.kostat.go.kr) 참조]    4. 공장부지면적 : 공장설립(예정)지 면적을 토지대 장상 면적(㎡) 근거로 정확히 기재 (지번별 합산 면적)
