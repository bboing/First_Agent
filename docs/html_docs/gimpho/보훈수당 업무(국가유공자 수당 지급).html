신청방법

관할 읍면동 보훈담당자에게 구비서류 제출하여 신청

교부방법

신청한 계좌로 매월 25일 입금

 

처리절차

국가유공자등 대상자가 관할 읍면동에 직접 신청& nbsp;-> 읍면동에서 접수 및 검토 후 시청 보훈담당자에게 제출 -> 심사 후,  매월 25일 대상자에게 보훈수 당 지급

처리방법

시청 보훈담당자가 새올행정시스템 맞춤형서비스(보훈)와 행복e음 시스템을 통해 대 상자 대조 · 확인하여 매월 25일 수당 지급  

업무설명

1. 보훈명예수당

 - 대상자 : 보훈 부 등록된 국가유공자 및 유족

 - 지급액 : 만75세이상 매월 10만원 / 만75세 미만 매월 8만원

 

2. 참전명예수당

 - 대상자 : 보훈부 등록된 만 80세이상 참전유공자

 - 지급액 : 6.25참전유공자 매월 10만원 추가지급 / 월남참전유공자 매월 5만원 추가지급

 

3. 독립명예수당

 - 대상자 : 보훈부 등록된 독립유공자 및 유족

 - 지급액 : 매월 20만원 추가지급

 

4. 참전유공자 배우자 수당

 - 대상자 : 사망한 참전유공자의 배우자 (단, 사망한 참전유 공자가 참전명예수당을 수혜자였던 경우에만 배우자 승계가 가능)

 - 지급액 : 매월 5만원

 

5. 경기도 생활보조 보훈수당

 - 대상자 : 보훈부 등록된 국가유공자가 중위 소득 50% 이하인 경우

 - 지급액 : 매월 10만원 추가지급

 

6. 국가유공자 사망위로금

 - 대상자 : 국가유공자의 유족

 - 지급액 : 15만원

 

주의사항
특이사항

- 보훈수당은 신청주의로 대상이 되어도 대상자가 신청하지 않으면 지급 및 소급 불가

- 국가유공자가 사망 또는 전출 시에는 사망 또는 전출한 달까지만 지급, 이후 지급 중지

- 국가유공자가 전입 후 신청시에 전입일 다음달부터 지급

- 참전유공자 배우자 수당은 참전유공자가 사망한 다음 달부터 배우자 수당 지급

관련법규

- 국가유공자 등 예우 및 지원에 관한 법률

- 김포시 국가보훈대상자 예우 및 지원에 관한 조례
