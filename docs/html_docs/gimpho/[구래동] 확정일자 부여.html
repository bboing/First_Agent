처리방법

 ※ 신고방법
 1. 청구자 : 전입된  주소지의 주택임대차계약서에 확정일자를 부여받고자 하는 자
 2.  청구기관 : 주민등록 전입처리가 된 주소지 관할 읍면동 또는 등 기소
 3. 청구방법 : 주택임대차계약서를 주민등록 전입업무담당에게 제시하고  확정일자 부여를 구두로 청구

업무설명

전입신고시 또는 재계약시 주택임대차 계악서 원본에&n bsp;확정일자를 부여함으로써, 세입자를 보호하는 업무

주의사항
특이사항

 - 원본 계약서에만 확정일자 부여 가능하며 사본은 불가함.

- 확정일자를 이미 부여받은 계약서에는 다시 부여받을수 없음.

    (단, 재계약 관련된 내용을+추가 변경하는 사항은 가능함)

  - 임대차신고제와 달리 확정일자 부여신청은 대리인도 위임없이 계약서 제출하면 부여가능함.
