처리방법

 ○ 처리방 법 : 가족관계등록부 폐쇄( 기본증명서에 사망일시, 사망장소, 신고인 기록)

 ○ 처리결과 확인 : 사망자의 기본증명서, 자녀 또는 배우자의 가족관계증명서, 혼인관계증명서

업무설명
 ○ 사망신고의 다양한 경우에 따라 제출서류, 과태료 등이 상이함.

   1 ) 무연고자의 경우
   2) 한국인이 외국에서 사망한 경우
&n bsp;  3) 외국인 가족이 국내, 국외에서 사망한 경우
   4) < /span>사망한지 오래되어 사망진단서(사체검안서) 재발급이 어려워 인우 보증 사망신고를 할 경우 등
 ○ 신고인에 해당하는 친족의 범위 : 8촌 이내 혈족과 4촌 이내 인척(민법제777조)

특이사항

 ○ 일반적인 사망신고를 제외한 경우는 민원여권과에서 안내

   - 일반적인 사망신고 : 한국인이 국내에서 사망, 사망진단서(사체검안서)가 있고 친족이 있는 경우

 ○ 사망과 동시에 또는  1년 이내 원스톱안심상속서비스 신청 가능(첨부문서 참조)
