 - 경찰서 사건접수 시 경찰관이 처리

 - 정보공개청구 : 10일

처리비용

 해당없음

신청방법

 

 - CCTV 영상은 개인정보에 해 당하므로 사건/사고 처리 시 경찰서에 신고하면 경찰관이 열람 후 사건처리

 

 O 개인이 개별로 CCTV 영상을 원할경우, 정보공개청구(www.open.go.kr)를 통해 신청 가능

 - CCTV 영상은 본인만 신청가능하며, 본인이 촬영된 영상만을 요구할 수 있음

 - CCTV 함체에 붙어있는 관리번호 확인 가능시 번호 기재(김포-A-000, 김포-P-000 등)

 - 날짜와 시간(5분이내)이 특정되어야 제공가능함

 - 본인을 제외한 주변 타인의 개인정보 영상은 모자이크 처리 후 제공

    - > 영상내 타인이 많을 경우 모자이크 작업시간이 많이 소요

 - 교통사고는 자동차등록증 첨부

 - 분실 등 수사가 필요한 사항은 비공개 대상

 - 수수료 발생(동영상 1G(기 가)마다 800원)

 

교부방법

 

 - 경찰서 사건접수 시 경찰관 처 리 후 안내

 - 개별 정보공개청구 시 정보공개포털을 통해 답변

 

처리절차

 

 - 정보공개청구는 10일이내 답변

 - CCTV 종류에 따라 해당 부서에서 답변

   -> 방범(도시안전정보센터), 불법주정차(교통과) 등

 

처리방법
