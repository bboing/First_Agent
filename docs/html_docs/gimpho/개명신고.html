업무설명

 ○ 개명신고는 사전에 반 드시 주소지 관할 법원의 개명허가결정절차를 거쳐야 함
   -주소가 김포인 경우 개명신청 : 인천가정법원 부천지원(032-320-1 114)

 ○ 개명허가대상자가 미성년자인 경우에는 미성년자의 법정대리인(친권자)가 신고

특이사항

 ○ 개명신고 처리 후 주민등록증, 운전면허증, 여권, 인감 등은 개별적으로 변경 신청을 해야 함
 ○ 학교, 은행, 회사, 보험사 등 해당기관에도 개명사실을 증명하는 < span style="line-height: 1.2; font-family: 맑은 고딕; font-size: 10pt; margin-top: 0 px; margin-bottom: 0px;">자료를 제출하여 개명사실을 소명
