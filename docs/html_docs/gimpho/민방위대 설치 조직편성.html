업무개요
민방위대조직은 지역민방위대, 직장민방위대, 민방위지원대(기술지원대)로 운영하고< br>민방위대편성은 1~4년차는 4시간교육 5년차부터는 민방위가 해제되는 40세까지는&nb sp;1시간 비상훈련을 받고 있음
민방위대 편성자원의 관리로  교육,&nbs p;훈련, 동원업무의 기초자료 활용 및 편성자원의 정확한 통계자료를 구축 하는 업무
