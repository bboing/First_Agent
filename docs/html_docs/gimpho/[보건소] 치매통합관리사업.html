처리절차

치 매관리팀 방문 → 1차 선별검사 → 2차 진단검진(치매의심 대상자) → 3차 감별겸사 (원인 확진)

처리방법

1. 선별검 사 결과(정상) : 치매예방교육 및 정기적인 검진서비스 제공

2. 선별검사 결과(인지저하자)

  - 진 단 / 감별검사 : 치매진단 및 원인 확진을 위하여 병원 연계 서비스 제공

  - 인지재활프로그램 참여(음악치료, 미술치료, 원예치료 등)

  - 치매환자 등록 관리

업무설명

○치매의 예방에서부터 조기발견, 치료 , 재활 진행 단계별 적정관리 등

 

1. 1차 선별검 진 : 주소지 관할 거주 만 60세 이상 주민 (소요시간 : 15~30분)

2. 2차 진단검사 : 1차 선별검진에서 인지저하로 결과가 나왔을 경우 치매신경심리검사(CERAD-K) 시행 후 협력의사( 신경과 전문의)와 검사결과 면담실시 (소요시간 : 1시간 정도)

3. 3차 감별검사 : 치매로 진단된 경우 뉴고려병원 또는 김포우리병원으로 연계하여 감별검사 시행 (소요시간 : 1~2시간 정도)

 

○통합적인 치매관리 서비스 제공

 

1. 인지프 로그램( 음악치료, 미술치료, 원예치료 등) : 정상 또는 치매 고위험, 치매어르신을 대상으로 현재 남아있는 기능을 보존 또는 강화하고 손상이 심한 기능을 보완하여 일상 생활에 필요한 기능을 향상 또는 유지시키기 위하여 프로그램 진행

2. 치매치료관리비 지원

  - 대상자 : 김포시 거주 만 60세 이상 치매치료약 복용자 중 기준 중위소득 140% 이하인 경우

  - 지원액 : 월 3만원(연 36만원) 한도 내 (< /span>신청일이 속한 월부터 발생한 약처방 또는진료비중 급여부분 본인부담금)

  - 지원절차

      ․ 대상자 : 북부보건센터 치매관리팀에 치매치료관리비 신청접수

  &n bsp;   ․ 치매관리팀 : 치매치료관리비 심사 후 국민건강보험공단에 대상자 통보 및 치료관리비 위탁

  &nbs p;   ․ 국민건강보험공단 : 대상자에게 매월 치료비 지급
