 외국인 체류확인서 열람 및 교부

업무개요

 전입세대열람확인서와 동일한 성격으로, 특정 건물 또는 시설의 소재지를 체류지로 신고한 외국인(임차인)의

 성명과 체류지 변경일자를 확인할 수 있는 서류(2023. 6. 14. 시 행)
