업무명

청년월세 한시 특별지원 사업

업무개요

청년층의 주거비 부담 경감을 위해 청년월세를 한시적으로 지원(신청기간 : ~ 2025. 2. 25 .)

접수처

장기본동행정복지센터

대상자

19~34세 이하로 부모님과 별도거주하면서

임차보증금 5천만원 및 월세7 0만원 이하 주택에 거주하는

무주택자

(* 청년본인 중위소득6 0%이하 & 원가구 중위소득100%이하인자)

신고기한

 

과태료

 

구비서류

1. 신청서

2. 임대차계약서

3. 청년 및 부모의 가족관계증명서(상세)

4. 최근3개월간 임차료 계좌이체 증빙서류 (보낸사람, 받는사람 명시 필수)

5. 통장사본

6. 청약통장 사본

처리기간

청년가구, 원가구 소득재산 조사 후 기준적합시 지원

처리비용

 

신청방법

- 온 라 인 : 복지로(https ://www.bokiro.go.kr) 또는 어플리케이션을 통해 청년 본인이 인증절차를 거쳐 신청 가능

- 오프라인 : 주소지 관할 주민센터에 방문하여 신청
