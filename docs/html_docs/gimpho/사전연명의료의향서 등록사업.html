처리 절차

방문 후 상담→신분증 확인→상담사와 의향서 설명 후 작성→등록자가 국립연명관리기관에 등록→등록카드발급(신청하는 경우)

처리 방법

업무 설명

사전연명의료의향서 : 자신의  연명의료중단결정 및 호스피스에 관한 의사를 직접 문서로 작성한 것

 

  ○ 상담절차

     연명 의료결정법 및 사전연명의료의향서 설명→ 작성 의사 확인

     → 작성자 신분증 확인→ 사전연명의료의향서 세부사항 설명 및 결정→ 사전연명의료의향서 작성(본인)

     사전연 명의료의향서 변경 및 철회 안내→ 연명의료정보처리 등록

    

  ○ 상담원칙

      신분 증 확인을 통한 본인 직접 작성, 상담자와 작성자 1:1 대면상담
