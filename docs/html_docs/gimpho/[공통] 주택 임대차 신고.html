업무명

 주택 임대차 신고

업무개요

 정확한 임대차 시세정보 제공을 위하여 주택의 임대차 계약 당사자가 신고기한 내에 계약내용 신고 의무화

접수처

 - 방문신고 : 주택 소재지 관할 읍면동 행정복지센터

 - 온라인 신고 : 부 동산거래관리시스템

대상자

 보증금 6천만원 초과 또는 월차임 30만원 초과하는 주택임대차 계약 당사자 (임대인/임차인)

신고기한

 주택 임대차 계약체결일로부터 30일 이내 

과태료

 100만원 이하

  (2025.06.01부터 해태기간 및 계약 금액에 따라 차등 부과)

 

*유예기간 : 20 21.06.01 ~ 2025.05.31

&n bsp;(과태로 유예기간 내에 신고 시 과태료 면제)

구비서류

 1. 신고의무자(개인) : 임대차계약서, 신분증

 2. 신고의무자(법인) : 임대차 계약서, 대표자 본인 신분증, 대표자 증명서류(법인 등기부등본 등)

 3. 개인의 대리 인 : 임대차계약서, 대리인 신분증, 위임자(신고의무자) 서명 또는 날인된 위임장, 위임자 신분증 사본

 4. 법인의 대리인(법인 직원 포함) : 임대차계약서, 대리인 신분증, 법인인감 날인 위임장, 법인인감 증명서 원본

 ※ 임대차계약서 사본 가능

처리기간

 즉시(인터넷 신고의 경우 업무시간 내 3시간)

처리비용

 없음
