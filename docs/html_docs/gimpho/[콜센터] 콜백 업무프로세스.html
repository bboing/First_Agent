1. 콜백이란?

  < /span>▶ 콜백이란?
   &nb sp; 민원인이 콜센터로 통화시도 했으나 연결이 안될 때 OB요청하여 번호를 남겨 놓은 걸 의미 합니다.

  1) 콜백 기본원칙
    ① 콜백은 100% 진행합니다 .
     ② 콜백 접수 되고 1시간 이내 처리하는게 원칙
     ③ 부재시 2차까지 진행 후 상담 완료

2. 처리시간

  1) 콜백 요청건은 콜백 접수되고 1시간 이내, 부재시 2차까지 진행 후 상담완 료
           ① 업무시간 내 콜백 접수 후 1시간 이내 OB진행 해야함
          ② 1차 OUT 부재라면 2차 OUT 시간차는 최소 30분후 진행
     < span style="line-height: 1.5; font-family: 맑은 고딕; font-size: 10pt; margin-top: 0 px; margin-bottom: 0px;">        (벨소리 후 소리샘으로 연결~ 멘트까지 진행)
        &n bsp; ③ 전일자 또는 주말 콜백건이 있을 경우 업무 시간 오전10시부터 진행
     &nb sp;       (전일자 또는 주말 콜백건은 오전 8:30~9:59까 지는 진행하지 않음)
  2) 콜백 처리 예시
          ① 10시 0분에 1차 OB시도,&n bsp; 10시 30분 2차 OB시도 (O)

4. 콜백 요청후
재인입

   1) 시민이 콜백에 대한 언급을 하는 경우 예시
       ① 조금 전에 전화를 했는데 안받아서 ARS로 번호를 남겨놨는데요. 기다리다가 전화를 했어요~
       ② ARS로 번호 남겨 논게 있었는데 전화를 못받아서 부재중보고 제가 전화했어요~
    
&n bsp;  2) 콜백 시민 재인입 시 처리 방법
      상담사 : 반갑습니다. 김포시콜센터 김포수 입니다. 무엇을 도와 드릴까요?
      ; 시   민 : 부재중 전화가 와서 연락 드렸어요.
      상담사 : 아, 그러십 니까? 전화번호 말씀해주시면 확인 후 안내 도와 드리겠습니다.
      시   민 : 010-1234-5678이요
      상담사 : 잠시만 기다 려 주시겠습니까? (민원인검색 / 이력으로 히스토리 확인)
        &n bsp;        기다려 주셔서 감사합니다. 시민님이 저희 센터로 콜백 요청 하셔서,
        &n bsp;        해당건으로 연락 드린걸로 확인되고 있습니다.
        &n bsp;        문의사항 말씀해 주시면, 확인 후 안내 도와 드리겠습 니다.
   
     [시민의 문의에 맞는 처리방법으로 업무처리 진행]
    상담사 : 시민님~ 센터로 전화요 청주신, 콜백사항에 대해서는 추가로 연락 드리지 않아도 괜찮으십니까?
    시   민 : 네, 괜찮습니다. (상담이력 : 콜백 건 추가 연락 원치 않음으로 기 재 必)
