· 산 모의 의식불명, 장기간(4주이상) 입원, 유선손상 등으로 모유수유 불가 진단을 받은 경우
(단, 영양플러스사업·선천성대상이상 환아관리 사 업의 조제분유 지원과 중복 불가)          &n bsp;
