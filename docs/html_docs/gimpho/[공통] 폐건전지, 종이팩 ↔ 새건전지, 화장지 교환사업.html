업무명

 폐건전지, 종이팩 ↔ 새건전지, 화장지 교환사업

업무개요

 

 ＊ 폐건전지(20개 또는 0.5kg) ↔ 종량제 봉투(10L 2매 혹은 20L 1매) 또는 새 건전지(3개 / ※ AA, AAA 선택)

     - 교환대상 :  산화은전지, 알칼리/망간전지, 리튬1차 전지, 니켈∙카드뮴 전지

     ;    ※ 차량/군용 배터리 및 리튬2차 전지는 별도로 배출

 

 ＊ 종이팩 ↔ 화장지 교환

     - 200㎖ 100매당 1롤 (우리가 일반적으로 알고 있는 가정용 1롤임)

     - 500㎖   55매당 1롤 (우리가 일반적으로 알고 있는 가정용 1롤임)

     ; - 1000㎖ 35매당 1롤 (우리가 일반적으로 알고 있는 가정용 1롤임)

 ＊ 종이컵 (사이즈 무관) 250개 ↔ 화장지 1롤 교환


 - 종이팩, 종이컵은 내용물을 비우고 물로 헹군 후 펼쳐서 건조

 - 종이팩, 건전지  자체 수량 파악 후 주민센터 방문 → 확인 후 즉시 교환

 

- 제외대상 : 전해액이 흘러내리 는 폐건전지
