※임대조건신고를&nbs p;하지않거나 거짓으로 신고시 처벌

표준임대차계약서로 처음 신고 하는  ;경우에는 '임대차계약 신고'에 해당하며 이때 금액이 5%상한선의 기준이  ;되며 이때 신고하는 임차인의 입주일이 임대의무기간 시작일임.

때문에  ;첫 신고때 임대료가 매우 낮은 경우는 추후 임대차계약 변경 신 고시 5%상한선에 제한을 받기 때문에 조심해야함.

공실인 경우는 따로  신고할 것은 없음.

임대료 상한 5%는 계약건당 최소 1년 이상을 말하며, 2년계약이라고 해서 10%가 아님.

임대사업자 등록전에&nbs p;작성한 일반임대차 계약서를 표준임대차계약서로 다시 작성할 의무는 없음.
나중에 만기일이 오면 그때 표준임대차계약서로 신고 하면 됨.
