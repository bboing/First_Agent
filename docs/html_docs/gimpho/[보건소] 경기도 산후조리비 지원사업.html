업무명

경기도 산후조리비 지원사업

업무개요

산모신생아 건강보호, 출산가정의 경제적 부담 완화 및 아이 낳기 좋은 환경 조성

접수처

- 출생등록하는 읍면동 행정복지센터
   (타 주민센터 신청 불가)

- 정부24 온라인 신청

대상자

1. 지원대상 : 부 또는 모가 다음의 요건을 모두 충족하는 경우

&nbs p;- 출생일 및 신청일 현재 경기도에 주민등록을 두고 있을 것

 - 신청일 현재 실제 도에 거주하고 있을 것

 - 신청일 현재 영아의 출생일이 12개월이 지나지 않았을 것

 - 경기도에 출생 등록이 되어야 함

 - 부모 중 한명 이 반드시 대한민국 국적 소지자여야 함

 - 다문화가정의 외국인 및 영주권자는 외국인등록 사실 증명으로 갈음


2. 예외지원 : 부부모두 외국인이거나 배우자확인이 어려운 외국인

 - 출산자(모)의 체류자격이 F-5(영주)일 것

 - 출생일 및 신청일 현재 경기도에 거주하고 있을 것

 - 신청일 현재 영아의 출생일이 12개월이 지나지 않았을 것

 

3. 지원금액 : 출생아 1인 당 50만원(지역화폐) 지원

        ;            (쌍둥이 100만원, 세쌍둥이 150만원)

 

 

신청기간

출산일(포함) 기준 12개월 이내

과태료

없음

구비서류

신분증, 주민등록등초본, 출생증명서, 가족관계증명서

처리기간

신청일로부터 14일 이내 지급

처리비용

없음

신청방법

- 본인(배우자)가 자녀 주민등록상 행정복지센터 방문신청, 정부24 온 라인으로 구비 서류 첨부 신청

교부방법

- 신청서 작성후 담당자 심사후 지급 결정

- 출생아 1 인당 50만원의 김포페이(모바일) 지원

   (쌍둥이 100만원, 세쌍둥이 150만원 )

처리절차

해당 읍면동 행정복지센터 신청, 접수  →  담당자(보건소) 상담, 조사&nbs p; →  신청일로부터 14일이내 지역화폐로 지급

처리방법

1. 신청서 접수 및 상담·조사 후 대상자 결정
2. 휴대폰에 김포페이 앱 설치 및 회원가입
3. 지역화폐 발급 안내 문자 발송

4. 지역화폐 발행

업무설명

1. 지역화폐 사용방법(대형가맹점, 대형프랜차이즈, 유흥업소, 온라인업체에서는 사용 불가)
2. 사용기한 : 지역화폐 발급 후 3년 이내 사용

주의사항
특이사항

* 지역화폐 발행일로 3년 경과시 자동 소멸

관련법규

모자보건법, 저출산고령사회기본법, 경기도 저출산고령사회 대응과 지속발전을 위한 조례, 경기도 산후 조리비 지원사업 조례
