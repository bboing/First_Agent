업무개요

처리비용
 
&nb sp;
 
신청방법

 ○ 「정부24」사이트(http://www.gov.kr) 인증서 로그인 후
 ○ 검색창에 '여권 재발급' 입력 조회하여 사진파일 업로드하고 수수료 납부 후 신청
  - 온라인 신청 시 해 당 링크의 여권사진 규격을 반드시 확인
  - 사진의 규격이 맞지 않는 경우 여권신청이 반려될 수 있음 (반려 시 모든 수수료 환 불)

  - 사진반려 시'「정부24」서비스 신청내역’에서 직접 환불 신청한 후 처음부터 다시 신청하셔야함
   ※ 해외거주자는 영사민원24(http:/ /consul.mofa.go.kr), 에서 신청, 모든 재외공관에서 수령 가능 

 
 
 
교부방법

○ 온라인 신청 시 지정한 여권수령기관에 본인이 직접 방문하여 여권 수령

 - 신분증과 잔여기 간 남은 구여권이 있는 경우 반드시 지참

 
 
 
처리절차

1. 온라인여권 접수→ 2. 여권심사(본인이 지정한 여권수령기관)→3. 여권제작(조폐공사)→4.여권수령기관 조폐공사에서 여권 등기수령 후 육안심사 및 판독→ 여권신청인 지문 및 안면인식 확인 후 여권교부

 
 
 
처리방법
○ 온라인 접수 된 여권재발급신청서의 사진파일 규격 등 서류심사  후 조폐공사에서 제작된 여권 교부

업무설명
○ 온라인으로 여권 재발급&nb sp;신청을 할 수 없는 사람
 - 미성년자, 직전 여권 발급일 이후 개명 또는 주민등록번호 변경자
 - 병역미필자 , 현역, 대체복무자, 관용여권 또는 긴급여권 신청자
 - 전자여권 발급기록이 없는 사람, 여권 상습분실자 등
 & nbsp; ⇒ 여권상습분실자 : 잔여기간 남은 여권 소지자 중 신청일 기준 5년내 1회이상 분실자 
○ 수령안내 : 온라 인 신청 시 입력한 핸드폰번호로 문자 수신
○ 수령기관 : 주소지와 상관없이 온라인 신청 시 지정한 전국 여권사무대행기관(추후 변경불가)< br>○ 수령방법 : 본인이 직접 방문(신분증 및 잔여기간이 남은 구여권 반드시 지참)
 
 
 
주의사항
특이사항

○ 여권수령방법 : 신청인 본인이 직접 방문하여 수령(대리수령 불가)

- 잔여기간이 남은 구여권과 신분증 반드시 지참

○ 사진의 규격이 맞지 않는 경우 여권신청이 반려될 수 있음 (반려 시 안내 후 본인이 수수료 환불 신청)

 - 사진 파일크기가 200kb 이하이고 해상도가 300dpi 이상이어야 함

 - 6개월 이내 촬영된 배경이 흰색인 사진으로 여권용 사진 규격에 부합되어야 함

 - 휴대폰 셀카사진, 지나친 포토샵으로 보정한 사진 불가
