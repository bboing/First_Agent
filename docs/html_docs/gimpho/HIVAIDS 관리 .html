업무설명

 

감 염인 진료비 지원: 실명으로 등록된 관내 감염인(건강 보험가입자)은 에이즈 진료비 중 본인부담금 지원 받을 수 있음

감염인이 선 납부 후 환급해 주거나 의료기관 진료비 후불 협조 통해 보건소에서 직접 의료기관에 후납하는 방식으로 지원함

 

주의사항
특이사항

- 비급여로 청구된 치료제는 지급 대상이 아님

- 진료 비 중 진료와 무관한 전화사용료, 제증명료, 병실료 차액, 선택진료비 등 비급여에 해당하는 부분은 지급 대상 아님

- 기초수급대상자는 HIV/AIDS 관련 진료비 면제됨
