- 기준중위소득 150%이상(라형)

- 온라인 신청
  ․ 방 법 : 복지로(www.bokjiro.go.kr) 또는 핸드폰 복지로 앱-> 산모신생아 건강관리 -> 복지서비스 신청 하기
  ․ 기 간 : 출산예정일 40일 전부터 ~ 출산 후 60일까지
  ․ 서 류 : 가족관계증명서, 임신확인 서(또는 진단서) 첨부
