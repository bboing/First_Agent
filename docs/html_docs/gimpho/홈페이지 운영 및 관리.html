업무설명


※시홈페이지운영범위(25개 사이트)
-대표홈페이지(외국어홈페이지 포함, 영문, 일문, 중문)
-직속기관(평생학습,보건소,농업기술센터,맑은물 사업소,클린도시사업소,시립도서관 ,도시안전정보센터),읍면동
-특화사이트(열린시 장실,김포시의회,통합예약,문화관광, 뉴스포털,< /span>통합일자리정보,사회적경제지원센터)

 

※홈페이지 비회원 제도 운 영

-아이디, 패스워드를 이용한 회원제도 없음(비회원제 운영 )

-따라서, 회원탈퇴의 개념이 없음 , 저장되는 특정 개인정보가 없음

*저장되 는 개인정보는 이름, 인증 식별번호로 개인을 특정할 수 없는 정보
-게시판 글쓰기, 예약하기, 대형폐기물 결제 등에서만 비회원 로그인 (실명인증)하여 사용

※비회원 로그인(실명인증) 방법
-휴대폰 인증 : 본인 명의의 휴대폰이어 야만 가능, 명의자가 다를 경우 인증 불가( 아이핀 인증 이용해야함)

  ;              *인증기록이 남지 않음으로 홈페이지 로그인 필요시 매번 로그인 해야함

-아이핀 인증 : 미성년자 등 본인 명의의 휴대폰이 없는 경우 사용, 민간기업(나이스, 사이렌 등 신용평가

      & nbsp;        정보 회사에서 제공하는 서비스)에서 발급받아 인증

-네이버아이디 로그인 : 네이버 아이디를 소지한 자로써 최초 휴대폰 본인인증 < /span>1회만 하면 다음번에 별도 &nb sp;인증없이 로그인 할 수 있음

 

※홈페이지 담당자 및 연락처, 콘텐츠 자료 최신화 관리방침
-분기별 콘텐츠 집중 정비기간 운영(정기)

-정책 변경, 인사발령에 따라 콘 텐츠 정비(수시)
-페이지별 담당부서에서 자료관리(페이지별 담당부서,연락처, 최종수정일공개)
-홈페이지 자료내용 관련 문의는 페이지별 담당 연락처로 연결 요망
-고시공고 관련 문의도 해당 공고 담당부서로 연결(예전 공고 포함)

 

※통합예약 (강좌, 체험, 시설대관, 모집) 신청 관련

신청 오류, 결제 오류, 인증 실패 등 기술적 오류 및 문의는 홈페이지 담당자로 연락

예약 내용, 예약 취소, 결제 취소, 환불 등 전반적인 예약 관리는 해당 담당부 서로 연락

 

※대형폐기물 홈페이지 결제 서비스
-홈페이지를 통해 대형폐기물 결제 및 스티커 출력 가능(프린터 있어야함), 모바일 빼기앱도 사용 가능

-배출주소지 입력, 배출품목입력 =>수수료결제 =>스티커출력 =>수거처리
-스티커 출력 버튼 미생성시 평일 근무 시간 전화 요청

-기술적인 오류는 홈페이지 담당자에게 연락하고, 대형폐기물 품목 선택, 결제 취소, 환불 등은 자원순환과 대형폐기물담당자에게 연락(☎ 2765)

 

※시민자율광고란 게시판 관련

-시민자율광고란 게시판에 등록되는 글은 김포시와 상관없는 개인, 일반기업 등이 홍 보 목적으로 게시

-간혹, 김포시에서 운영하는 정책으로 오인하여 문의하는 경우가 있음, 문의는 해당 글에 연락처로 연락

하고 다만, 글에 포함된 외부 링크나 연락처 이용 시 유의가 필요하다고 전달 필요

&nbs p; *광고내용만 봐서는 건전한 광고인지 확인이 어려움
  *일자리가 필요한 시민에게는 “통합일자리정보홈페이지-채용정보”에서 민간, 공공기관 채용정보 확인이 가능함으로 안내 요청

 

※홈페이지 게시 차단 운영
-개인정보 또는 금지 키워드를 포함한 게시글은 솔루션에 의해 자동 게시차단


※홈페이지 이용 에 따른 PC 최적화 환경 : 인터넷 익스플로러외 브라우저 이용 권장(크롬, 엣지 등)
