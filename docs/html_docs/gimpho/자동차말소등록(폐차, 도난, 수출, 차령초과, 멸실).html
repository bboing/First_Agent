 자동차말소등록(폐차, 도난, 수출, 차령초과, 멸실)
업무개요
국내에서 등록된&nb sp;자동차가 폐차, 도난, 수출, 차령초과, 멸실 등의 사유가 발생한& nbsp;경우 자동차 소유자의 신청에 의해 기존의 등록효과를 
소멸시키는&n bsp;등록업무
접수처
전국 업무
※ 단, 멸실 : 소유주의 사용본거지 차량 등록사업소 
대상자
자동차를 말소하고자 하는 자
신고기 한
1.폐차 : 폐차증명서를 발급받은 날로부터 1월 이내
2.수출 :&n bsp;수출계약서에 수출하기로 명시된 날로부터 1월 이내
3.도난, 멸실 :  기한은 없음.
4.차령초과 : 기한은 없으나 늦게 신청할 시&nb sp;말소되는 시점까지 자동차세가 부과되는 문제가 있으므로 폐차입고증을 받게&nb sp;되면 즉시 신청하는 것이 좋음.
과태료
*폐차(최고금액 : 50만원) : 말소사유가 발생한 날로부터  1월이내.
신청기간만료일로부터 10일까지 : 5만원,
신청기간만료일로부터 10일이후 1일  초과시마다 : 1만원
* 수출
1월이내에 수출말소신청치 않을 시&nb sp;: 20만원
구비서류
1. 폐차말소 : 폐차 장에 먼저 차량을 입고 시킨 후, 폐차 시킨 이후 폐차장에서&n bsp;대행해서 말소하러 오므로, 직접 민원인이 방문할 필요 없음
2.&nb sp;수출말소 : 말소신청서, INVOICE, 수출업체 사업자등록증, 수출업체 인 감증명서(대표가 오면 생략가능), 위임장(대표가 오면 생략 가능), 신청자&nbs p;신분증, 자동차 등록증, 자동차 번호판 2개 
3. 도난말소 :  경찰서에 신고한 후 발급받은 도난신고확인서
4. 차령초과말소 :  폐차장에서 발행한 차량입고확인서
5. 멸실 : 멸실사실인정서(관할구청에서 발급)
※ 공통< /b>서류 : 신청서 및  자동차등록증 (대리인 신청시 : 자동차  소유주의 신분증과 도장 날인된 위임장 첨부하여 신청)
※ 차령초과 말소를 제외한  ;모든 말소는 압류와 저당이 없어야 신청 가능

교부방법
 
처리절차등록서류 접수 → 검토 (자동차등 록 출력 대조 및 확인(압류,저당 등)) → 등록세 납부, 확인  → 말소 등록 → 말소사실증명서 발급
처리방법
※폐차,수출,도난,멸실 : 민원인이 신청하면&n bsp;담당자가 구비서류와 수수료등을 검토하고 말소등록처리하여 말소사실증명서 발급.
※차령초과 : 서류가 접수되면 1월 이내의 기간을 정하여 그 기 간내에 권리행사 등의 절차를 진행하도록 통지하고, 기간내에 권리행사 등의  절차를 진행하지 아니하는 때에는 당해 자동차의 소유자에게 폐차할&nbs p;것을 통보하고 말소등록함.
업무설명

1. 폐차  : 폐차업의 등록을 한 자에게 폐차요청하여 폐차인수증명서를 발급받아 신청 하는 말소 (가장 일반적인 말소임.) 폐차장에 문의.
2. 수출 :  수출하는 자동차
3. 도난 : 차량을 도난당했을 때 가까운&nbs p;경찰서에 신고한 후 도난신고확인서를 구비하여 말소하는 업무
4. 차령초과  : 차령이 11년 이상(승용자동차 기준)된 자동차 중 압류와  저당이 있지만, 환가가치가 없다고 판단하여 압류와 저당이 있는 채로&nb sp;말소시켜주는 제도. 차 량 실물이 있어서 폐차장에 입고시켜야 함. 차령초과 말소된다고 하여 압류와 저당이 사라지는 것이&n bsp;아님.
5.&nbs p;멸실 : 김 포시 관할 자동차만 가능. 오래된 차량 중 여러가지 사유로 차량의 실물이 없다고 판단되는 자동차만 신청  ;가능. 

멸실 신청-> 멸실 인정 -> 차량 압류, 저당  해제 -> 멸실 말소. 또는 멸실 신청-> 멸실 인 정 -> 이해관계자 통지 -> 멸실 말소. 

멸실 인정 및 말소는 각각&nb sp;약 한달 반 소요. 차령 12년 이상(승용차 기준)만 신청 가능.
