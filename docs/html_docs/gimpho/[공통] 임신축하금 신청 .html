업무명

 임신축하금 신청

업무개요

 

접수처

 1. 김포시보건소 (본관 1층)

 2. 주소지 관할 읍면동 행정복지센터

 3. 온라인 신청(정부24)

대상자

 - 신청일 이전 180일 이상 김포시 주민등록을 두고 거주한 임신부

   ( 외국인 임 신부는 신청일 이전부터 180일 이상 김포시에 외국인 등록을 두고

     거주한 임신부로 배우자가 대한민국 국적이어야 함)

 - 출생일 전까지 거주기간이 180일 충족되지 않는 경우

   신생아의 출생일로부터 180일 이상 김포 시에 주민등록을 두고 거주한 경우

신청기한

 지급대상 요건을 충족한 날부터

 6개월 이내

과태료

 없음

구비서류

 ㅇ 임신부 신청시 : ① 신분증   ② 임신 확인서 (산부인과 병의원)    ;③ 임신부 명의 통장사본 

< span style="line-height: 2; font-family: 맑은 고딕; font-size: 10pt; margin-top: 0px ; margin-bottom: 0px;">     &nb sp;              ;       ④ 임신부의 등본 또는 초본 1부 (주소변동내역 포함)

     ※ 외국인 임신부인 경우 추가 서류 : 외국인등록사실증명서 , 가족관계증명서

 

 ㅇ 대리인 신청시 : 임신부 신청시 서류 + 대리인 신분증, 가족관계증명서 추가 지참

    &nb sp;(※ 대리 신청인의 범위 : 임신부의 배우자, 임신부의 친부모 및 시부모)

 

 ㅇ 임신부 명의 통장 외 대리 인 통장으로 임신축하금 수령을 원하는 경우  : 해당 신청 서류 + 위임장 추가 지참

 

  ;     EX)  임신부 신청 (임신부 통장 수령) : 임신부 신청 서류

        ;     대리인 신청 (임신부 통장 수령) : 대리인 신청 서류

        ;     대리인 신청 (대리인 통장 수령) : 대리인 신청 서류 + 위임장 (미리 작성해와야 함)

    &nb sp;       임신부 신청 (대리인 통장 수령) : 임신부 신청서류 + 가족관 계 증명서 + 위임장 (방문작성)

        ※ 위임장 다운로드 : https://www.gimpo.go.kr/health/contents .do?key=4514
