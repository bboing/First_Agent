(지급해지)

 - 3년간 통장유지 + 교육(3년간 총 10시간) + 자금사용계획서 제출

 - 정 기 또는 타 보장급여 확인 조사를 통해 근로‧사업소득이 유지기준 초과하고, 자립역량교육 이수 + 자금사용계획서 제출 시

(환수해지)

 - 3년 만기 후 지급 사유 발생하였으나 교육이수 기준 미달< br> - 자금사용계획서 미제출 시

 - 정기 또는 타보장급여 확인조사를 통해 근로활 동이 없는 가입자로 확인될 경우
 - 사전 적립중지 신청 없이 본인적립금 누적 12개월 미납 시
 - 교육이수 기준 미달 시
 - 압류, 가압류 시
 - 본인 사망 시
 - 본인 요청 시
