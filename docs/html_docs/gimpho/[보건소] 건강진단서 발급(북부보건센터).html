과태료

 

구비서류

- 내국인 : 신분증(운전면허증,주민등록증, 여권(여권+여권정보증명서 또는 여권+주민등록등본) / 청소년증 ,학생증  (학생증에 주민등록번호가 없는경우+주민등록등본)&n bsp; 

- 외국인 : 외국인등록증   

 *모든신분증 원본지참 (핸드폰사진, 사본불가)

 

처리기간

4일 (주말,공휴일제외)

처리비용

기본검사(폐결핵,매독)12,180원,  (기본+B형간염추가)19,520원,

(기본+요검사추가)13,090원, (기본+B형간염, 요검사 추가)20,430원, (폐결핵X-ray)10, 340원, (폐결핵X-ray+B형간염검사)17,680원,  최초원본발급 무료
추가발급:방문시1통300원(신분증원 본지참)/온라인발급시 무료

신청방법

신분증 지참후 북부보건센터 민원실 방문하여  제증명  신청서 제출 -> 수납후접수 

교부방법

- 본인이 오는 경우 신분증 지참후 직접교부

    대리방문시 영수증 뒷장의 위임장(본인이 서명한 위임장), 위임자와 검사자 신분증 모두 지참해야 함  

- 공인인증서 나 본인휴대폰인증시 온라인발급&nb sp;가능 (발급처: 인터넷 검색창 "공공보건포털" 검색) 

처리절차

접수후-> 검사실검사-> 방사선실검사 -> 결과판정 -> 4일후 교부

처리방법

검사항목: 폐결핵, 성병검사(매독) 검사 후 이상&n bsp;유무 판정 후 교부
(B형간염 검사 추가시: 항원 항체검사  /요검사(4종) )

업무설명

1. 근무시간 : 월~금, 오전 9시~오후  6시(중식 12~13시) ※ 휴일,중식시간 검사불가.

 

2. 학교기숙사나 일반회사 입사시(별정, 계약, 임시직,파트타임제)  건강진단서 제출을 요할때

 

주의사항
특이사항

-주소지나 근무자와 관계없이 가까운 김포시보건소, 북부보건센터 이용가능 (김포시 다른 보건지소 불가)       &n bsp; 

- 검사가 필요한 경우 오전 9시~11 시40분, 오후13시~17시40분까지 방문 요망(검사시간 및 대기인원시간까지 고려) &nb sp;

- 모든 신분증 원본지참 필수 (사본이나 휴대폰&n bsp;사진불가)      

- 재발급의  경우, 유효기간이 남아있어야하고, 검사받았던 기관에서만 가능 (방문시원본신분증지 참) / 온라인재발급가능 

- 대리방문시 본인이 서명한 위임장과 영 수증, 위임자 신분증과 검사자 신분증 모두 지참후 방문    & nbsp;  

- 검사결과 유소견자의 경우, &n bsp;김포시 보건소(사우동)에 방문 재검사후 발급 (발급이 지연될수 있음)&nb sp; 

- 외국인이 출입국 관리사무소에 외국인 등록증을 발급 받고자 할경우는& nbsp;김포시 보건소로 안내해야함 (외국인결핵확인서 발급불가)
