처리방법

 

업무설명

- 신규발급 : 이전에 문화누리카드 발급한 내역 없는 자

- 재발급 : 이전에 문화누리카드 발급한 내역이 있으나, 카드분실 및 카드 유효기간 만료 자

- 재충전 : 이전에 문화누리카드 발급한 내역이 있고, 유효한 카드 소유하고 있으나, 잔액 없는 사람

 

- 합산신청 : 가구 구성원별 카드 각각 발급에 따른 1장의 대표카드에 카드 합산 요청< /p>

  * 동일세대만 합산 가능하며, 합산신청도 카드 발급 서류와 동일

특이사항

- 문화누리카드는 문화생활(영화, 도서, 여행, 체육, 공예품)에 이용가능한 바우처 카드 이며

   생활용품 및 식음료 등 구입에 이용 불가

 

 *문화누리카드 홈페이지 가맹 점 확인 가능

 

- 바우처 금액(14만원) 사용기한 은 연도 내 이며, 연도내 사용하지 않은 건에 대하여는 자동소멸되므로
