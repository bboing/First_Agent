업무설명
소나무류 생 산업자가 반출금지구역이 아닌 지역에서 재배 또는 생산한 소나무류 조경수,  분재, 굴취목, 원목을 사업장 이외의 지역으로 이동하는 경우  소나무류 생산확인 신청서를 작성하여 제출해야 하며, 담당직원의 현장 육안 검사 또는 관련 연구기관(국립산림과학원)의 현미경 정밀검사 실시 후 이상 이 없을 때 소나무류 생산확인증을 즉시 발급
