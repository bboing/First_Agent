[건설기계의 정의]

☞ 건설기계란 건설공사에서 사용할 수 있는 기계로서 대통령령으로 정하는 것을 의미


  &nbs p; 건설기계 관리법 시행령 [별표 1]에서 정하고 있는 건설기계의 범위는 아래와 같음
    1.불도저, 2.굴삭기, 3.로더, 4.지게차(단,전동식으로 솔리드타이어를 부착한 것 중 도로가 아닌
    장소 에서만 운행하는 것은 제외), 5.스크레이퍼, 6.덤프트럭(적재용량12톤이상인 것,
    단 화물차량 제외), 7.기중기, 8.모터그레이더, 9.롤러, 10.노상안전기, 11.콘크리트뱃칭플랜트,
    12.콘크리트피니셔, 13.콘크리트살포기, 14.콘크리트믹서트럭, 15.콘크리트펌프, 16.아스팔트 믹싱
    플랜트, 17.아스팔트피니셔, 18.아스팔트살 포기, 19.골재살포기, 20.쇄석기, 21.공기압축기,
  &nb sp; 22.천공기, 23.항타 및 항발기, 24.사리채취기, 25.준설선   가. 지게차 (전동지게차 제외, 전동지게차는 일반적으로 물류장비로 분류)
 &nb sp;   ->도로(도로교통법상 도 로)에서 운행하지 않는 전동지게차는 등록의무 없음
      ->도로에서 운행하려는 전동지 게차는 필요서류 있으면 등록 가능(제작증,제원표,형식승인번호 등)

    나. 덤프트럭(적재용량 12톤이상) 
    -> ; 일반적으로 흙·자갈 등을 운반하는 차량
     ;    ※ 화물을 운반하는 차량은 자동차관리법에 의한 자동차로 분류

 

[건설기계 신규 등록 필요서류]

1.건설기계등록 신청서
2.양도증명서
3.위임장(인감날인, 법인 및 대리인 제출시)
 - 인감증명서
4.법인이면 등기부등본
5.사업자등록증-사본 (사업용인 경우, 소유용인 경우 필요 없음/사용본거지 주소입력용)
6.건설기계 제원표
7.건설기계 제작증(국내)
 - 수입 건설기계 : 수입면장 등 수입사실을 증명하는 서류(수입자의 무역고유번호 부여증 또는 사업자등록증),

         & nbsp;            &nb sp;    건설기계 형식승인서(신고수리서) 또는 동일형식 수입 신고서
8.차대번호 각자 탁본 2부
9. 종합보험 가입증명서 (9종, 해당 건설기계에 한함)덤프트럭/기중기(타이어식)/콘크리트믹서트럭/트럭적재식 콘크리트펌프/트럭적재식 아스팔트살포기/굴삭기(타이어식)/트럭지게차/도로보수 트럭/노면측정장비
10.국민주택채권확인서
11.세금계산서
12.영업용이면 대여업 계약 서류 - 건설기계임대차계약서
13. 배출가스 인증서 또는 인증생략서
14.소음인증서 또는 인증생략서(건설기계8종)
 - 덤프트럭, 콘크리트믹서트럭, 콘크리트펌프, 굴삭기(19~500kw), 로더 (19~500kw), 천공기, 항타 및 항발기
15.정부수입인지(3,000원)
16.수입증지 : 4,000원

[건설기계 이전등록 필요서류]

1. 건설기계 이전등록 신고서
2. 건설기계등록증(검사증)- 원본
3. 양도를증명하는서류(양도증명서, 양 도인 인감 날인, 양수인 도장(또는 인감) 날인)
 -> 양수인이 법인 시 인감도장 날인
4. 양도인 인감증명서 ("3개월" 이내)
5. 양수인 사업자등록증 사본(사업용), 법인이면 등기부등본(말소사항포함)
6. (개인) 본인 -> 신분증
   ; (법인,대리인) 위임장(인감날인,대표자 본인이면 필요없음), 인감증명서, 대리인 신분증
7. 대여업체 이전 등 동의서(해당되는 경우만, 영업용)
 -> 대여업체 인감증명서 첨부("3개월 이내)
8. 보험가입증명서류(9종)< br>덤프트럭/기중기(타이어식)/콘크리트믹서트럭/트럭적재식 콘크리트펌프/트럭적재식 아스팔트살포기/굴삭기 (타이어식)/트럭지게차/도로보수트럭/노면측정장비
9. 정부수입인지(3,000원)
10. 수입증지 1,000원
11. 번호판(영업용->자가용/자가용->영업용)
 -> 1 0일 이내 반납
     ※ 10일 초과 시 3만 원, 1일 초과시마다 1만원 가산하여 최고 40만원
12.도시철도채권매입필증 : 서울 , 부산, 대구, 대전, 광주 (해당 지자체로 이전하는 경우)
 -> 김포시 내 은행에서 발행이 불가하여 해당 지자체에서 이전등록 처리하라고 안내
