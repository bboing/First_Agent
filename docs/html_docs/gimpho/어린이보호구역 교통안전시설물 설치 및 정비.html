ㅇ 어린이보호구역 지정 = 유치원 원 장 및 초등학교 교장이 어린이보호구역 지정신청서를 작성하여 김포시청 교통 과 어린이보호구역담당자에게 제출 -> 김포시청 교통과에서 관할경찰서에 요청& nbsp;-> 경기남부지방경찰청에서 지정 (-> 자치구에 통보)

ㅇ  ;어린이보호구역내 교통안전표지판을 설치 => 유치원 또는 초등학교 주출입문으 로부터 반경 
    300m 범위안에서 자치구에서 설계용 역하여 경기남부지방경찰청 규제심의후 설치
