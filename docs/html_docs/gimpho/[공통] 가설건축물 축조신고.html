 - 철근콘크리트조 또는 철골콘크리트조가 아닐것

 - 존치기간이 3년 이내 일 것

 - 전기,수도,가스 등 새 로운 간선공급시설의 설치을 필요로 하지 아니할 것

 - 전시를 위한 견본주택

 - 컨테이너 또는 이와 비슷한 것으로 된 가설건축물로서 임시사무실,임시창고,임시숙소로 사용되는 것< /p>
