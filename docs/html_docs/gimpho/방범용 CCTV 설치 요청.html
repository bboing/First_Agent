신청방법

 

 민원콜센터에서 설치요청 내용을 받 아 도시안전정보센터 담당자에게 업무 이관 

 

 ( 확인사항 )  

 1. 민원인 성명  2. 전화번 호  3. 설치요청지 상세 주소(**사거리 앞, **슈퍼 앞)  4. 설치요청 사유

 

교부방법

 

 ( CCTV별 관리부서 )&nbs p;

 방범(도시안전정보센터), 불법주정차(교통과), 재난(안전총괄과) 

 도로관리(도로관리과), 쓰레기투기(자원순환과)

 

처리절차

 

 1. 방범용CCTV 설치요청 시 담당자가 접수하여 방범CCTV 설치 후보지로 등록

 2. 예정지에 대하여 김포경찰서에서 우범 지역, 범죄 발생, 치안 사각지대 등 빅데이터 분석

 3. 후보지에 대하여 김포시 통합관제위원회에서 우선순위 최종 결정

 4. 당해년도 예산범위에서 방범CCT V 설치

 

처리방법

 

 - 민원콜센터에서 설치요 청 내용을 받아 담당자에게 업무 이관

 

  ;- 김포시 통합관제위원회에서 결정된 우선순위대로 방범 CCTV 설치
