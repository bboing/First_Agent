 위생용품제조업 품목제조보고
업무개요
 위생용품제조업 영업자(일부 제품)가 시,군,구에& nbsp;품목제조보고하는 민원사무
 ※ 품목보고 대상 품목 : 세척제, 헹굼보조제, 일회용 기 저귀·팬티라이너(의약외품 제외), 식품접객업소용 물티슈
접수처
 온라인(식품안전나라&nb sp;https://www.foodsafetykorea.go.kr/main.do)
대상자
 위생용품제조업 영업 중 품목제조보고 대상제품 생산하는  업소
신고기한
 제품생산 시작 전이나 제품생산 시작후 7일 이내
과태료
 미보고 시 건당 40만원
구비서류

처리기간
 즉시
처리비용

교부방법
 위생용품관리법령 위반 ·저촉사항 검토(사용가능 원료 여부, 기준규격 위반 여부 등) 후&nbs p;품목제조보고 승인
처리절차
 위생용품 품목제조보고서 작성 → 온라인 접수&nb sp;→ 서류검토 → 승인(결재)
처리방법
 - 위생용품관리법령 위반·저촉사항 검토(사용가능 원료&nbs p;여부, 기준규격 위반 여부 등) 후 품목제조보고 승인
 -&nb sp;식품안전나라에서 승인된 품목 확인(출력) 가능
업무설명
 - 품목제조 보고 대상인 제품별로  보고하여야 함
 ※ 품목보고 대상 품목 : 세척제, 헹굼보조제, 일회용 기저귀·팬티 라이너(의약외품 제외), 식품접객업소용 물티슈
 - 제품에 유통기한을 표시하는 경우: 품 목제조보고서에 내용 기재
주의사항
특이사항
 1. 제품 및 성분의&n bsp;적정성여부 검토 후 반려가 될 수 있음 
 2. 혼 합원료는 혼합된 개별 성분의 명칭을 모두 기재
 3. 배합비율&nb sp;표시는 위생용품 공전(「위생용품의 기준 및 규격」에 사용기준이 정해져&nbs p;있는 원료 또는 성분에 한함)
 4. 세척제, 헹굼보조제는 위생 용품 공전 [별표1,2] 참조하여 사용할 수 있는 성분만 가능
