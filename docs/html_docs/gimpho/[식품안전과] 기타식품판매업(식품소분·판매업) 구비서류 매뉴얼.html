업무 명

식품소분·판매업 신고
(식품소분업/식용얼음판매업/식품자동판매기영업/유통전문판매업/집단급식소 식품판매업/기타식품판매업)

업무개요

식품소분·판매업 영업자가 시,군,구에 신고하는 민원사무

접수처

 김포시 식품안전과
(단, 식품조사처리업, 식품등수입판매업은
지방식품의약품안전청에 허가(신고), 축산물가공업은 경기도청 허가, 식육포장처리업 및 판매업은 김포시 축산과 신고)

식품소분·판매업 영업자

신고기한

 영업개시 전

행정처분

 무신고(등록) 영업시 고발조치

(등록-5년  이하의 징역 또는 5천만원 이하의 벌금
 신고-3년 이하의 징역 또는 3천만원 이하의 벌금)

구비서류

&n bsp;1. 교육이수증(한국식품산업협회 온라인 교육 가능, 식품자동판매기는 한국휴게음식업중앙회에 온라인 교육 가능)
 2. 건강진단결과서(유통전문판매업은 제외)
 3. 수질검사(시험)성적서(수돗물이 아닌 지하수 사용시) 또는 상수도 요금 영수증 사본(식품자동판매기인 경우)
 4. 신고인신분증(주민등록증, 운전면허증, 외국인등록증, 여권)
 5. 대리신고
   - 위임장
   - 위임자의 인감증명서, 법인인감증명서 및 법인 등기부등본(법인에 해당)
   - 대리인신분증(주민등록증, 운전면허증, 외국인등 록증, 여권)

처리기간

즉시 (3시간이내) 처리비용 수수료 : 28,000원
 면허세 : 종별 부과(18,000원~45,000원)

신청방법

&nbs p;신고인 출석신고 및 민원24 이용가능(공인인증서 필수)

교부방법

처리절차

신고: 신고서작성 → 접수 → 검토 → 결재 → 신고증발급  ※ 시설조사가 필요한 경우에는 신고수 리후 15일이내 실시

처리방법

※ 신고시 확인사항
 1. 건축물대장등본(일반/집합) : 건물용도 근린생활시설
 2. 해당장소에 대한 이중신고 확인여부
 3. 행정처분사항 및 관허사업제한(체납) 조회

 2. 영업신고증 수령후 관할 세무서 방문

특이사항

 1. 업소 전산자료 조회를 위해 자치구 담당자와 상담요함

 3. 타법 저촉여부 신고인 직접 확인
