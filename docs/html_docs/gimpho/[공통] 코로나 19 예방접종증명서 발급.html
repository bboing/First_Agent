코로나19

행정나 눔

기타

업무명

&n bsp;코로나백신 예방접종증명서 발급(수정)

업무개요

 코로나백신 접종 내역 확인 이 필요한 경우 증명서를 발급해주는 업무

접수처

 김포시 읍면동 행정복지센터

&nb sp;정부 24 누리집

대상자

 국문, 영문 예방접종증명서가 필요한 자

신고기한

 필요시(상시)

과태료

 없음

구비서류

1. 국문 :  예방접종증명서를 발급받을 자의 신분증

2. 영문 :  예방접종증명서를 발급받을 자의 신분증 (영문 성명 확인 필요하므로 가급적 여권 지참 권유)

처리기간

 신청즉시

 (온라인 접수의 경우 근무시간 내 3시간)

처리비용< br>

 없음

신청방법

 읍면동 행정복지센터 민원실

교부방법

 종이 또는 스티커

처리절차 및 방법

 신분증 확인에 따른 즉시 발급

업무설명

&nb sp;1. 방역패스(접종증명서) 발급받을 사람의 신분증 필수 지참.

 2. 김포시 거주자가 아니여도 본인 혹은 대리 신청 가능(전국 발급)< /p>

 3. 내국인, 외국인 모두 발급가능(단, 한국에서 코로나백신 접종했을 경우에만 가능)

 4. 미성년자 예방접종증명서 발급시,

 &nb sp;   법적보호자 혹은 등본상 같은 가구원이 ①신분증 ②가족관계증명서 혹은 등본 2가지 지참시 가능

5. 영문예방접종증명서 발급 : 여권 지참(주민번호 기입된 여권) 또는 신분증 & nbsp; 

    여권만료자는 질 병관리청 시스템에서 성함이 영문으로 변환이 되지 않아 수기로 입력해야하므로

    발급받을 분의 성함을 민원인(발급요청자)이 영문성함을 ①메모해주거나 ②여권을 가지고 와야 함.

6. 여권 미지참 : 메모로 영문 성함을 작성해 줄 경우 여권에 있는 영문성함과 동 일해야 함.

     &nb sp;              ;   성함 오기, 오류로 인한 책임은 민원인에게 있음 (온라인 신청 시 오타 유의)

7. 종이 로 발급되는 서류는 코로나백신 접종사항이 모두 나오지만 스티커 발급은 4차까지만 출력됨.

&n bsp;   동절기예방접종(5차)는 출력이 되지 않음

주의사항
특이사항

 1. 내국인/외국인 관계없이 외국에서 접종시 보건소에서 사전등록 후 보건소에서 '예방접종확인서' 발급요청 필요

     (행정복지센터에서는 외국에서 접종한 경우 시스템에 접종확인 되지 않음)

 2. 스티커 발급시, 신분증에 스티커 부착까지 법적 의무사항이 므로 신분증 지참 필수
