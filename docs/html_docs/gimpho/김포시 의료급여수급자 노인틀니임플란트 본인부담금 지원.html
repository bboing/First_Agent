업무설명< br>

 

1) 대상자: 만 65세 이상 의료급여수급자 (건강보험 혜택을 받는 자는 지원 불가)

&n bsp;             &nb sp;(신청일 기준 김포시 1년 이상 주민등록을 두고 실제 거주 확인)

 

2) 시술 전 치과에서 의료급여 틀니•임플란트 대상자 등록

 

3) 시술 종료 후 60일 이내 주소지 행정복지센터 방문신청

 

4) 지원기준

& nbsp;  ○ 틀니

      - 급여횟수: 7년에 1회

     - 상악, 하악 별 도 지원

    

&nb sp;  ※ 예외) 7년 이내 틀니 재제작을 원하는 경우: 의료급여 틀니 등록 신청서, 의사소견서 제출 - 내부 검토 - 재제작 가능 여부 안내

 

 

   

   ○ 임플 란트

     - 급여횟수: 평생 2개

     - 임플란트 치료재료 포함하여 지원

     - 완전 무치악은 지원 제외

   ;   - 상•하악 구분 없이 어금니, 앞니 모두 적용

의료급여수급자의 경우

 - 틀니 본인부담: 1종 수급권자 5%, 2종 수급권자 15% 발생

 - 임플란트 본인부담: 1종 수급권자 10%, 2종 수급권자 20% 발생
