◎ 음식물류 폐기물 분리·배출 방법

   • 이물질을 제거하고 물기는 최대한 제어하여 전용봉투 또는 수거용기에 물이 고여 있지 않도록 배출

   • 소금성분이 많은 된장·고 추장·간장·김치 등은 헹구어 배출

    • 이쑤시개, 나무젓가락< span style="line-height: 12pt; letter-spacing: -1pt; font-family: 굴림; font-size: 10pt; margin-top: 0px; margin-bottom: 0px;">, 비닐·병뚜껑·패 각류 등은 재활용을 어렵게 하는 물질로 반드시 제거 후 배출

   • 음식물류 폐기물이 아닌 물질은 종량제봉투로 배출
