업무설명
1. 건축물의 건축 또는 공작물의 설치(옹벽, 태양광발전시설 등)&nb sp;
 가. 건축물의 건축의 경우는 건축허가(신고)의제 처리되므로 건축허가 (신고) 신청과 동시에 개발행위 신청
       서류 제출
 나. 주거지역, 상업지역, 공업지역 이외에 단독주택,  근린생활시설, 농가용 창고 등의 입지와 임야상 경사
       도가 도시계획 조례 이상으로 높을 경우 도시계획위원회&nb sp;심의 대상임
 다. 공작물의 설치 중 태양광발전시설의 경우 건 물 위에 설치할 경우 개발행위 대상은 아니며 토지에 설치
  ;     할 경우 농지(산지)전용과 도시계획위원회 심의를&nb sp;거쳐 처리됨
 라. 옹벽의 경우 도시지역, 비도시지역을 구분하여&nbs p;무게를 산정하여 허가 여부를 검토하여 처리
2. 토지의 형질변경은&nbs p;대부분 건축물의 건축으로 신청되어 처리
3. 토석의 채취 : 우 리시는 대상 지역이 없음
4. 토지분할 
 가. 최소분할 면적 이상으로 분할할 경우 가능하며 우선 개발행위허가를 득한후 한국국토정보공사에&nbs p;분할측
      량을 의뢰하여야 함.
 &nbs p;나. 농업진흥지역(절대농지)의 경우 최소분할 면적은 2천㎡임.
  다. 신청서는 개발행위팀에서 교부받을수 있음. 
5. 녹지지역, 관리지역 또는 자연환경보전지역에 물건을 1개월 이상 쌓 아놓는 행위  
  가. 야적장 조성 목적의 토지의&nbs p;형질변경에 대한 개발행위허가와 준공 처리후 다시 개발행위허가를 신청하
