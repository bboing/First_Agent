  <구호대상>

- 이재민 : 재해를 입은 사람

- 일시대피자 : 재해가 예상되어 일시 대피한 사람

- 심리회복지원 대상자 : 이재민, 일시대피자, 재난을 직접 목격한 사람, 재해현장에서 구호 자원봉사 또는 복구활동에 참여한 사람

- 외국인 : 국내 거주 외국인도 구호대상에 포함(불법체류자 제외)

 

 <구호종류>

- 임 시주거시설 제공

- 급식, 식품, 의류, 침구 그밖의 생활필수품 제공

- 심리회복 상담, 정신보건시설 연계 등 심리회복 지원 등

 

 <구호기간 >
