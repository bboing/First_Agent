영업자가 시,군,구에 신고하는 민원사무

접수처

 김포시 식품안전과
(단, 식품조사처리업, 식품등수입판매업은

축산물가공업은 경기도청 허가,

식육포장처리업 및 식육판매업은 김포시 축산과 동물방역팀 신고)

무신고(등록) 영업시 고발조치

(등록-5년& nbsp; 이하의 징역 또는 5천만원 이하의 벌금
 신고-3년 이하의 징역 또는 3천만원 이하의 벌금)

 

구비서류

 1. 교육이수증(한국식품산업협회 온라인 교육 가능)

 2. 건강진단결과서(구 보건증)
 3. 제조방법설명서(제조·가공업/즉석판매제조· 가공업 해당)
 4. 시설사용계약서(식품운반업 차고 또는 세차장 임대할 경우)
 5. 수질검사(시험)성적서(수돗물이 아닌 지하수 사용시)
 6. 신고인신분증(주민등록증, 운전면허증, 외국인등록증, 여권)
 7. 대리신고

   -위임자의 인감증명서, 법인인감증명서 및 법인 등기부등본(법인에 해당)
  ;  -대리인신분증(주민등록증, 운전면허증, 외국인등록증, 여권)

처리기간

 식품제조가공업, 식품첨가물 제조업 : 3일
 즉석판매 제조가공업 등 : 즉시 (3시간이내)

처리비용

수수료 : 28,000원
 면허세 : 종별 부과(18,000원~45,0 00원)

신청방법

 신고인 출석신고 및 민원24 이용가능(공인인증서 필수)< br>

교부방법

신분증 확인(대리가능) 및 면허세 납부확인후 교부

처리절차

등록:  등록신고서 작성& nbsp; → 접수 → 현장확인 → 검토 → 결재 → 등록증발급 
신고:  신고서작성 → 접수 → 검토 → 결재 → 신 고증발급  ※ 시설조사가 필요한 경우에는 신고수리후 15일이내 실시

처리방법

※ 신고시 확인사항
 1. 건축물대장등본(일반/집합) : 건물용도 근린생활시설 

 ※ 식품제조가공업 및 식품첨가물 제조업의 경우 근린생활 시설(제조장), 공장 가능
 2. 해당장소에 대한 이중신고 확인여부
 3. 행정처분사항 및 관허사업제한(체납) 조회

업무설명

 1. 식품제조판매업 영업을 하고자 하는 자가 해당 영업소 소재지 관할 구청에 신고 하는 민원사무
 2. 영업신고증 수령후 관할 세무서 방문
 <업종설명>
 1. 식품제조ㆍ가공업 : 식품을 제조·가공하는 영업
 2. 즉석판매제조ㆍ가공업 : 식품을 제조ㆍ가공 업소내에서 직접 최 종소비자에게 판매하는 영업
 3. 식품운반업 : 부패ㆍ변질되기 쉬운 식품 위생적으로 운반하는 영업(다만, 당해 영업자의 영업소에서 판매할
                        목적으로 식품을 운반하는 경우와 당 해 영업자가 제조ㆍ가공한 식품을 운반하는 경우를 제외)
 4. 식품소분ㆍ판매업
   가. 식품소분업 : 식 품 또는 식품첨가물의 완제품을 나누어 유통을 목적으로 재포장ㆍ판매하는 영업
   나. 식품판매업
 &nbs p;     (1) 유통전문판매업 : 직접 제조ㆍ가공하지 아니하고 타인에게 의뢰하여 제조ㆍ가공된 식품 또는< br>              ;             & nbsp;        식품첨가물을 자신의 상표로 유통ㆍ판매하는 영업
&nb sp;      (2) 집단급식소 식품판매업 : 집단급식소에 식품을 판매하는 영업
        (3) 기타 식품판매업 : 300제곱미터이상 백화점ㆍ슈퍼마켓ㆍ연쇄점등에서 식품을 판매하는 영업
 5. 식품냉동ㆍ냉장업 : 식품을 얼리거나 차게 하여 보존하는 영업(다만, 수산물의 냉동ㆍ냉장을 제외)
 6 . 용기ㆍ포장류제조업
    가. 용기ㆍ포장지제조업 : 식품 또는 식품첨가물에 직접 접촉되는 용기ㆍ포장지를 제조 하는 영업
    나. 옹기류제조업 : 식품을 제조ㆍ조리ㆍ저장할 목적으로 사용되는 독ㆍ항아리ㆍ뚝배기등을 제조하는 영업
