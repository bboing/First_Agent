버스를 이용하는 승객이 일상적인 교통민원을&nbs p;신고하면 운수업체의 의견진술을 받아 위반사항을 과징금 처분 및 행정처 분

교부방법
국 민신문고 접수 시 처리절차 및 계획 안내. 처벌결과 문의는 2개 월 후 별도 유선문의  필요
