업무명
  건물번호 부여·변경·폐지 신청
업무개요
  건축물 등의 신축·개축 등으로 건물번호를 부여 또는 건물번호를 변경하는 경우 기존 건물번 호 분할이나 합병, 폐지하는 업무
접수처
 토지정보과 주소정보팀
대상자
 소유자, 건축주, 설계자, 시공사, 공사 감리자, 가설건축물 축조 신고자
신고기한
 건축물 사용승인 전
과태료
 건물번호판 설치 후 건물번호판이 훼손 또는 망실되었을 때 재설치하지 않으면 최대 50만원
구비서류
 건축물 개요, 건축물 배치도 (가설건축물의 경우 가설건축물 축조 신고 필증, 건축물의 위치를 표시한 도면)
처리기간
  14일 이내(휴일·공휴일 제외)
처리비용

없음

신청방법

 ○ 김포시청 민원동 3층 토지정보과 주소정보팀으로 첨부서류 구비하여 방문(대리인의 경우 도장 필요)

 ○ 정부24에서 신청자격이 있는 공인인증서로 들어가 첨부서류 등록 후 접수

교부방법
  신청인에게 건물번호와 건물번호판 규격을 문자메세지로 통보 (건물번호판은 교부하지 않음)
처리절차
  건물번호 부여 신청서 접수→건물번호 부여→건물번호 정리 결의→건물번호  부여 결과 통지→공고(고시)
처리방법

 

업무설명

 

 - 건물번호 부여: 건물 등을 신축·재축하는 경우, 30일이상 거 주나 정착하여 활동하는 구조물

 - 건물번호 변경: 건물 등의 증축·개축 등으로 주된 출입구 변경, 주소 사용자의 편의를 위해 건물번호 변경이 필요한 경우

 - 건물번호 폐지: 소유자 또는 점유자의 거주활동의 종료 등으로 건물번호를 사용할 필요가 없는 건물< /p>

 - 건물번호판의 제작·설치에 관한 비용은 건축물 등의 소유자 가 부담해야 함

특이사항

 ※ 부여받은 건물번호판은 지체없이 설치해야함
