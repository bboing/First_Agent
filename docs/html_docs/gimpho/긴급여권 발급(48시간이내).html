업무개요

 긴급한 사유로 급히 출국하여야 하는 경우 여권을 신속하게 발급받고자  신청하는 경우로

 48시 간내 발급여권(10년 전자복수여권)과 비전자단수여권이 있음

 O 48시간 내 발급여권 : 전국 여권사 무대행기관, 외교부 여권민원실

 

 O 비전자단수여권 : 광역 여권사무대행기관, 외교부 여권민원실, 인천공항민원센터, 서울시 강 서구청, 일산동구청, 부천시청, 파주시청 등 48개 기초지자체대행기관

(김포시청 불가)
