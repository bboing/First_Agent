업무명

 출산 서비스 통합 신청

업무개요

 김포시에 출생신고를 득한 아동의 출산서비스를 보건소를 대신하여 통합처리 지원

 (경기도 산후조리비 / 출산축하금 / 첫만남이용권 / 영아 (양육)수당 / 아동수당 등)

접수처

 1. 주소지 관할 읍면동 행정복지센터

< span style="line-height: 2; font-family: 맑은 고딕; font-size: 10pt; margin-top: 0px ; margin-bottom: 0px;"> 2. 온라인 신청(정부24)

대상자

 

1. 경기도 산후조리비 : 출생일 및 신청일 현재 경기도 주민등록을 두고 거주한 보호자

2. 출산축하금 : 둘째아 이상, 출생일 기준 180일 이전부터 김포시 주민등록을 두고 거주한 보호자

      & nbsp;            &nb sp;또는 출생일로부터 180일 이상 김포시 주민등록을 두고 거주한 보호자(자녀는 주민등록상 동일세대) 

       &n bsp;            → 김포시 6개월 거주요건 반드시 충족해야 지급

3. 첫만남이용권 : 2022년 1월 1일 이후 출생아 

4. 영아수당 : 2022년 1월 1일 이후 출생아 ( 만 1세까지 지원)

 * 양육수당 : 20 21년 12월 31일 이전 출생아/ 2022년 1월 1일 이후 출생아 부터는 만 2세 아동(만 6세까지 지원)

5. 아동수당 : 출생 월 ~ 만 8세 미만까지 지원

신 청기한

1. 경기도 산후조리비 : 출생일로 부터 12개월 이내 

2. 출산축하금 : 출생일로부터 6개월 이내

3. 첫만남이용권 : 출생일로부터 1년 이내

    & nbsp;            &nb sp;     (단, 출생로부터 1년이 사용기한이기도 하므로 신청주의)

4. 영아수당 : 수시가능

       & nbsp;         (단, 출생아 경우 출생일 기준 60일 이내 신 청하여야 출생월부터 소급지급가능)

 * < strong>양육수당 : 수시가능 (2022년 1월 출생아 부터는 만2세 도래 시 자 동 자격변경)

5. 아동 수당 : 수시가능

&nbs p;                 (단, 출생아 경우 출생일 기준 60일 이내 신청하여야 출생월부터 소급지급가능)

 

구비서류

신분증, 신청서 1부, 부모 중 1인의 주소변경이력이 포함된 등본 혹은 초본 1부, 통장사본 1부

처리기간

1. 경기도 산후조리비 : 2주 이내

2. 출산축하금 : 신청월의 익월 지급

3. 첫만 남이용권 : 30일 이내(연장 +30일가능)

4. 영아수당 : 30일 이내(연장 +30일가능)

< span style="line-height: 2; font-family: 맑은 고딕; font-size: 10pt; margin-top: 0px ; margin-bottom: 0px;"> * 양육수당 : 30일 이내(연장 +30일가능)

5. 아동수당 : 30일 이내(연장 +30일가능)

신청방법

  1. 김포시 보건소 또는 주소지 관할& nbsp;읍면동 행정복지센터 방문 신청

& nbsp; 2. 온라인 신청 (정부24)

  ; ※ 영아(양육)수당 및 아동수당은 복지로 온라인 신청도 가능

  ※ 읍면동 행정복지센터를 통해 신청하는 경우 간편하게 일괄 신청 가능함

교부방법

1. 경기도 산후조리비 : 50만원 김포페이  지급

2. 출산축하금 : 둘째아 부터 100만원 현금 지급(신청 시 제출한 계좌로 익월말 지급)

3. 첫만남이용권 : 국민행복카드로 200만원 바우처 포인트 지급

4. 영아수당 : 매월 25일 30만원 현금지급(신청 시 제출한 계좌로 지급 만 2세 미만까지 지급)

     * 양육수당 :

      ;   - 2021년 12월 이전 출생아 (출생일~11개월 20만원/12개월 ~ 23개월 15만원/ 24개월 ~ 만 6세 10 만원)

         - 2022년 1월 이후 출생아는 만 2세부터 10만원

5. 아동수당 : 매월 25일 10만원 현금지급(신청  시 제출한 계좌로 지급, 만 8세 미만까지 지급)

처리방법

 위 절차 에 맞춰 각 사업부서 검토 완료 후 적합 대상자에게 교부일 및 교부 방법에 따라 지급

업무설명

 출생신고 후 출산정책에관한 정부지원 서비스를 통합처리 받을 수 있도록 원스톱 서비스 지원

주의사항
특이사항

★ 신청 후 지급 시까지 신청인과 지원대상 아동의 주소를 확인하여 다른 시군구로 전출 하였을 경우 지급 대상에서 제외

   (출생아의 출생일 기준부터 지급될 때까지 김포시 주민등록 거주자만 지급 됨)

 ★ 외국인 지원 불가

    : 부 또는 모 둘 중 한명이 대한민국 국적으로 주민등록을 두고 있는 경우 신청가능 (신청인 : 한국국적자) 

 ★ 재혼 가정의 경우 실질적으로 양육하는 자녀 수 합산 (등본상 함께 거주)
