신청방법

O 해당 공무원 재직기관에서 관용여권&nb sp;발급 요청 공문 외교부 발송 후 출국 3개월 전 대상자가 신청

O 공무 원의 가족 중 성인은 출국 3개월 전 본인이 직접 방문신청 해야함

O 신청자 본인 및 대리수령, 등기우편 수령[(여권접수 시 착불(4,200-4,500원) 등기신청 가능]

  - 본인 수령(미성년자도 가능) : 접수증, 신분증(본인 수령 시 접수증 없이 신분증만으로도 교부 가능)

  - 대리 수령(미성년자도 가능) : 위임장(위임자 도장이나 서명 포함< span style="line-height: 1.2; letter-spacing: 0pt; font-family: 굴림; font-size: 1 0pt; margin-top: 0px; margin-bottom: 0px;">), 위임자 신분증(사본가능 ),

     대리인 신분증

O 1.신 청서 접수 → 2. 신청서 심사(외교부) → 3. 여권제작(한국조폐공사)  → 4. 여권배송(우체국) → 5. 여권수령 대사 및 판독&n bsp;→ 6. 교부

업무설명

O 외교부에서 여권신청 사 항을 심사하는 관계로 일반여권보다 발급기간이 2-3일 더 소요됨

O 공무원 본인은 5년관용여권이 발급되 며, 가족들은 파견기간+6개월의 관용여권이 발급됨

O 공무 여행기간이 최소 3개월 이상인 경우만 가족 동반이 가능함

O 관용여권 신청 시에만 공무원이 공무원증을 제시하는 경우 신분증으로 대용할 수 있다  

O&nb sp;관용여권 재발급시 유효기간이 6개월 미만인 경우에만 재발급 가능함(귀국일 기준)

  &n bsp;다만, 체류 비자(VISA) 취득 등의 부득이한 사유가 있을경우 공문에 명시되어 있어야 함.

O 문의는 민원여권과 여권팀(031-980-270 0~2702) 또는 외교부 여권헬프라인(02-733- 2114)로 안내
