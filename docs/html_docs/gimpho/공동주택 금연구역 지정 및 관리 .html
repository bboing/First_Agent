업무개요

 흡연을 예방하고, 비 흡연자를 보호하기 위한 환 경 조성을 위해 공중이용시설의 금연구역이 준수되도록 지속적인 홍보와 지도단속

신고기한

 

과태료

★ 금연구역에서 흡연한 경우 : 5만원

(공용주택의 복도, 계단, 엘리베이터, 지하주차장에서 흡연한 경우)

구비서류

1. 해당 공동주택의 세대주 명부에 관한 서류, 신청서

2. 금연구역 지정 동의서 또는 공동주택 세대주 2분의 1 이상이 금연구역 지정에 동의함을 입증하는 서류(공동주택의 복도, 계단,

엘리베이터 또는 지하주차장의 구분에 따라 동의한 서류)

3. 해당 공동주 택의 도면에 관한 서류(건축물 대장에 평면도)

4. 해당 공동주택의 복도(공용), 계단(공용), 엘리베 이터 또는 지하주차장의 내역에 관한 서류

처리기간

 

처리비용

 

신청방법

공동주택 거주 세대 중 2분의 1 이상이 그 공동주택의 복도, 계단, 엘리베이터, 지하주차장 4곳 전부 또는 일부를 금연구역으로 지정하여 줄것을 신청하면, 금연구역으로 의무지정할 수 있음.
